@js4y/Dialog

Introduction

Dialog is a tiny dependency-free javascript library built on a dialog element with minimal configuration, yet sufficiently variable.

Features

  • closing the dialog using the escape key
  • closing the dialog by click on the backdrop or element with the corresponding data attribute
  • loading external styles
  • preventing scrolling of the page when displaying the dialog
  • simple configuration and api
